This is my 3rd IT project that I was the last one to do while training at Kosta, and it is a project based on Spring Boot.
It is a project that used Thymeleaf as a front, not based on Jsp & Servelt learned during training, and even completed distribution using AWS.

One of the topics in the IT industry is the so-called 'war of saving developers'. The rapid rise of artificial intelligence after AlphaGo and the advancement of platform companies such as Kakao,
And IT, with expectations for the future of metaverse, which includes expectations for the blockchain sector represented by cryptocurrency and NFT, etc., In particular, the software sector is again emerging as a key competitive edge in the future.¬†Developers' capabilities required by companies are increasing day by day, and developers are feeling the need for self-development in line with this.
Of course, I can study by myself. However, in line with today's time when collaboration is important, Jambo is aiming for a website that allows IT researchers to be available. It also provides a community where developers can share various IT information.

üëæ My Role

  • User Join function
    • Spring Security
    • When Users register their Information, they can ID duplicate check function by AJAX.
  • Study Board
    • User can decorate their posts using SummerNote API when writing study board posts
    • User can close their study posts
    • User can see closed study posts look different on the view page depending on the status of the post list
  • Normal Board
    • Post registration, modification, and deletion functions.
    • Like Button (Only Users can do it, and User can do it only once in the same post.)
    • Post reporting system (only members can report in the same article, members cannot report in duplicate)
    • Only User can write posts by Spring Security.
  • Admin Page
    • Admin check the number of posts written by Users, Users registration, Users information, number of posts reported, and number of study posts on the administrator's main page
    • Admin can check the reason for reporting posts reported by Users
    • Hiding posts reported by Users from the boards and activation functions
    • Administrator Account Creation with Spring Security
    • Write notice board, competition board, and event board posts boards on the administrator page
  • AWS Deploy
    • AWS EC2 Instance Amazon Linux
    • Connecting the Instance by ssh, build & run
